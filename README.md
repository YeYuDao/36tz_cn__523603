<br/></br>### 课程介绍:
React.js 是 Facebook 推出的一个用来构建用户界面的 JavaScript 库。 然后当底层的数据变了，React 会自动处理所有用户界面的更新。 当数据变化了，React 概念上是类似点击了更新的按钮，但仅会更新变化的部分。 事实上，通过 React 你唯一要做的事情就是构建组件。
